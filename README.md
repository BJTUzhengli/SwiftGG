# SwiftGG

####【这是什么】
[SwiftGG](http://swift.gg) 的 iOS 客户端，简单收纳了网站里面的归档文章，方便大家浏览。

####【可能会有的问题】
这个客户端使用 JavaScript 实时抓取网站的方式，获取了文章的标题、链接等信息，所以如果 SwiftGG 网站改版，可能会造成客户端短期无法更新内容的情况，届时我会更新代码。如果您有能力，也可以自行手动修改相关代码。
